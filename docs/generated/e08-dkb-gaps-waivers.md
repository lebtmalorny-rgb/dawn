## Review rules

- A row can leave this register only after an external owner provides evidence and `docs/11_DKB_TRACEABILITY.md` is updated.
- A formal waiver must include approval identity, scope, expiry and compensating controls outside this repository.
- Local E08 evidence narrows risks but does not replace IAM, PKI, SIEM, SecMan, storage, registry, SELinux or network owner evidence.
- ДКБ-69 remains a conflict for Python backend until an explicit exception is granted.
